for loops : repeatable IF statements

for (initialization ; condition ; iteration){
statements
}
initialization: starting point

condition : ending point, it needs to be related to initialization

iteration : co-related to Condition, must be capable of changing the condition to false



JAVA Control Statements
1. Selection Statements
      - if ...else
      - switch ---case
2. Iteration Statements
      - for loop
      - while
      - do while
3. Jump statements
        - break
        - continue
