Escape sequences: MUST be given with double quote
    \n: starts a new line
    \t: paragraph space(tab)
    \\: single back slash
    \": double quote


    ----------------------------------------------------------
    Primitive Data Types:

    declare variable:
    		dataType  variableName = data;
    	       int i1 = 5;
    	       int i2;
    	       i2=10;


    		numerical primitives:
    					double > float > long > int > short > byte


    primitives:
    	int: for integers
    	double: for decimals
    	char: for single character '' single quote
    	boolean: true/false

    none primitive:
    String: sequence of characters, String of text, "" double quote
        String myName = "ihsan";

       PrimitiveCasting
       		smaller primitives can be directly assigned to larger primitive type
       		(implicit casting)(Widening Casting)

               larger primitives can NOT be directly assigned to smaller primitive types (need to be casted manually)
               (explicit casting) (Narrowing Casting) (Data Lost) (Veri kaybi soz konusu)

       		sout  : prints to console
            soutv : prints the name of the variable and the value in a format, useful to print the values
