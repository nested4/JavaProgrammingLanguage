  Unary Operators:
        	++: increase the value by 1.
        	-- : decreases the value by 1.

        	Post vs Pre:
        		Pre : changes the value immediately
        				++ a
        				-- a

        		post: changes the value in the second step.
        				a ++
        				a --

        Assignments operator          Shorthand operator
